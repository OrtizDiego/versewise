**VerseWise** is a modern, feature-rich Bible study application designed to bridge the gap between ancient texts and modern accessibility. Built with performance and user experience in mind, it leverages cutting-edge web technologies to provide a seamless reading and study environment.

## üöÄ Features

*   **Multi-View Interface:** Compare different Bible translations side-by-side, including original Greek and Hebrew texts.
*   **Deep Lexicon Integration:** Click on original language words to instantly view definitions, parsing information, and concordance data.
*   **AI-Powered Insights:** (Experimental) Logic integrated for generating summaries and insights using Google's Genkit AI.
*   **Sleek & Accessible UI:** A fully responsive interface built with Radix UI and Tailwind CSS, featuring dark mode support, smooth transitions, and keyboard navigability.
*   **Smart Navigation:** easy-to-use chapter and verse selectors.

## üèÉ‚Äç‚ôÇÔ∏è Getting Started

Follow these instructions to get a local copy of the project up and running.

### Prerequisites

*   Node.js 18+ installed on your machine.

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/versewise.git
    cd versewise
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Set up Environment Variables:**
    Create a `.env.local` file in the root directory and add necessary keys (Supabase URL, Anon Key, Genkit config, etc.).

4.  **Run the development server:**
    ```bash
    npm run dev
    ```

5.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
